# FindMyHomeUK - Estate Agent Client-Side Application

FindMyHomeUK is a React-based web application that allows users to search for properties, view details, and manage their favorite listings.

## Features

- Property search with various filters
- Detailed property listings
- Drag and drop functionality for favorites
- Image gallery for property photos
- Interactive property details page

## Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14.0.0 or later)
- npm (v6.0.0 or later)

## Plugins Used

This project uses the following plugins:

1. react-router-dom
2. react-dnd
3. react-tabs
4. react-image-gallery
5. react-datepicker
